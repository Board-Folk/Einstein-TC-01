# Einstein TC-02Mini 80 Column Adapter

![Built TC-02Mini v1 adapter board in Green](https://github.com/Board-Folk/Einstein-TC-01/blob/main/images/TC02Miniv1_Small.png)

A simple, non-replica (schematics the same) remake of the TC-02 80 column board. Could be mounted on a ribbon cable, or as pictured on PC104 pin sockets (at full stretch). Required Character ROM at I4 in CHARROM folder. Please see the original manual for instructions for use.
